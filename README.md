# 💧 Smart Water Distribution AI Dashboard

An AI-powered system for intelligent monitoring and prediction of water demand and infrastructure health.

---

## 🚀 Features

* 📈 **LSTM Water Demand Prediction**
* 🚰 **AI Leak Detection System**
* ⚙ **Automatic Valve Control Logic**
* 🌦 **Weather Impact Analysis**
* 🏗 **Infrastructure Health Monitoring**
* 📊 **SCADA Style Monitoring Dashboard**
* 🚨 **Critical Alert System with Sound**
* 📍 **City Based Monitoring**

---

## 🧠 AI Models Used

1. Random Forest – Water demand prediction
2. LSTM – Demand forecasting
3. Leak Detection Model – Pipeline anomaly detection

---

## 📊 Dashboard Modules

* System Overview
* Demand Analytics
* Weather Impact
* Infrastructure Monitoring
* AI Insights
* City Monitoring
